00038 /* BRP+1 = Fpclk/(CANBitRate * QUANTAValue)
00039    QUANTAValue = 1 + (Tseg1+1) + (Tseg2+1)
00040    QUANTA value varies based on the Fpclk and sample point
00041    e.g. (1) sample point is 87.5%, Fpclk is 48Mhz
00042    the QUANTA should be 16
00043         (2) sample point is 90%, Fpclk is 12.5Mhz
00044    the QUANTA should be 10 
00045    Fpclk = Fclk /APBDIV
00046    or
00047    BitRate = Fcclk/(APBDIV * (BRP+1) * ((Tseg1+1)+(Tseg2+1)+1))
00048 */
